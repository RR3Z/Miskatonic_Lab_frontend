# Палитра Miskatonic Lab

Источник палитры - текущие CSS-токены в `src/app/globals.css`. Основной визуальный образ: архив Мискатоника, старая бумага, латунные печати, глубокая сепия вместо чистого черного.

## Правила применения

1. Не использовать чистый `#000000` как основной фон. Для глубины проекта лучше работает `#161412`, а для краев и оверлеев - `#100E0C`, `#050403` и прозрачный черный.
2. Латунь должна быть акцентом, а не заливкой всего интерфейса. Хорошие места: лого, hover ссылок, small CTA, иконки, focus ring.
3. Бумажная карточка должна жить на темном фоне. Ее сила в контрасте `--ml-surface-paper` + `--ml-ink-on-paper`.
4. Для больших темных поверхностей использовать радиальный градиент и легкий noise, как в `LandingBackground`.
5. Для ошибок не уходить в чистый красный. Проектный danger - темная бордовая печать `#8B211F`, а для текста ошибок в Clerk - более читаемый `#E06A61`.
6. В светлой теме не ставить мелкий белый текст на `#7A7650`. Для кнопок использовать `#665F38`.

## Быстрые примеры

```tsx
<section className="bg-(--ml-bg-page) text-(--ml-ink-primary)">
  <a className="text-(--ml-accent-brass) hover:text-(--ml-accent-brass-strong)">
    Open case
  </a>
</section>
```

```tsx
<article className="border border-[var(--ml-border-aged)] bg-(--ml-surface-paper) text-(--ml-ink-on-paper)">
  <p className="text-(--ml-ink-on-paper-muted)">Case notes</p>
</article>
```

```css
.miskatonic-glow {
  box-shadow: 0 0 24px color-mix(in srgb, var(--ml-accent-brass) 14%, transparent);
}
```
