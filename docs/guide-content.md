# Справочник сыщика

Игровой текст, названия разделов, подписи схем и UI-копирайт справочника находятся в
`src/data/guide/guide.ru.json`. Это единственный источник русской локали для маршрутов
`/guide` и `/guide/<section>`.

## Структура данных

- `sections` содержит восемь разделов, их slug, символ, описание и блоки.
- Каждый блок имеет постоянный `id`, который используется в каталоге, anchor-ссылке
  и результате поиска.
- `ui` содержит подписи каталога, шапок, поиска, пустого результата и примеров.
- Формулы, примеры и схемы — данные блока, а не текст внутри изображения.

Для новой локали создайте отдельный JSON с той же схемой. Компоненты не должны хранить
пользовательский текст в JSX.

## Поиск и ссылки

Pure-функции находятся в `src/components/guide/utils/`:

- `guide-search.util.ts` — поиск по заголовкам, основному тексту, деталям, формулам
  и схемам;
- `guide-route.util.ts` — anchor-ссылки каталога и URL результатов.

Выбор результата ведёт на `/guide/<section>?q=<query>&match=<block>#<block>`.
Страница прокручивается к блоку, а Motion временно подсвечивает его фон.
